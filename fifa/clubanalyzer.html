How to install Paletools Club Analyzer

Paletools Club Analyzer es un bookmarklet, un marcador que en lugar de apuntar y dirigirte a un sitio web, ejecuta codigo javascript en el cliente que va a modificar a la WebApp de Ultimate Team, para ejecutarlo lo que hay que hacer es cualquiera de los siguientes dos métodos:

Paletools Club Analyzer is a bookmarklet, a bookmark that instead of redirecting you to a web site, it executes javascript code in the client side that will modify the Ultimate Team WebApp, in order to install it you have to do the following:

  1. PASO 1:Asegurarse que la barra de marcadores del navegador este visible, en caso de que no lo este hacer lo siguiente
    1. Ir a las opciones del navagedor > Barra de Marcadores > Mostrar Barra de Marcadores (CTRL + SHIFT + B en Chrome)
  2. PASO 2:Arrastrar el link de instalación a la barra de marcadores
  3. PASO 2b:En caso que arrastrar el link no funcione realizar lo siguiente:
    1. Presionar el boton DERECHO del mouse sobre el link de instalación
    2. Tocar en "Copiar dirección de enlace"
    3. Presionar el boton DERECHO en la barra de marcadores
    4. Tocar en "Añadir p´gina"
    5. Aparecerá un dialogo donde hay que ingresar el Nombre y la URL
    6. Donde dice Nombre, escribir Paletools
    7. Donde dice URL en el campo de texto presiomos el boton DERECHO del mouse y tocamos en "Pegar"
    8. Presionamos en GUARDAR
  4. PASO 3:Abrir la WebApp y clickear en el link que acabamos de agregar a la barra de marcadores
  1. STEP 1:Make sure the browser's bookmarks bar is visible, if its not then do the following
    1. Click on the browser settings button > Bookmarks > Ir a las opciones del navagedor > Barra de Marcadores > Show Bookmarks Bar (CTRL + SHIFT + B on Chrome)
  2. STEP 2:Drag the installation link to the Bookmarks Bar
  3. STEP 2b:In the event dragging the link does not work the follow the next steps:
    1. RIGHT click over the installation link
    2. Click on "Copy link address"
    3. RIGHT click on the bookmarks bar
    4. Click on "Add site"
    5. A dialog should appear where you can enter the Name and the URL
    6. Where it says Name, input Paletools v2.0
    7. Where it says URL on the input box RIGHT click and then click on "Paste"
    8. Click on SAVE
  4. STEP 3:Open the WebApp and click on the link we have just added to the bookmarks bar
